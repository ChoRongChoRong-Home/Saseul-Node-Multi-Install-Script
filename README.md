## Getting Started

First, run the shell script :

```bash
# Set basic variables set.
vi install.sh
:wq
# Permission settings
chmod 777 install.sh
# run
./install.sh
```

설치이후에는 saseul.sh 로 관리

```bash

chmod 777 saseul.sh
./saseul.sh

```
